I'm not sure if you can do it from a speeddial, but if you're calling an
outside number you could set the extension the speeddial calls to insert
pauses in the Dial() statement.

 

So, if you needed to call 555-5555 and then go to option 2, you could
have the following:

 

555555,1,Dial(Zap/g1/5555555wwwwww2)
