Hi all,

yes, that is so. The config file used to be in binary format for the  
old versions, even for the 79xx series. I have succeeded to create  
one by means of a hex editor and this wiki page:

http://www.voip-info.org/wiki/view/Configuring+Cisco+12SP+phones+with 
+Asterisk

For updating firmware, you will have to specify the image name in  
asterisk via the corresponding phone section sccp.conf (see examples  
or the source code).
