Hi,

I think the issue is "Skinny Phone Type  : unknow(365)" - what version
of chan_sccp are you using? The latest set from SVN has the 7921
supported and listed - I think that the issue is that chan_sccp does not
know the 7921 thus is only allows one line.

The bits below say it is unknown (type 365 is a 7921):

-- SEP001BD457F8B1: Building button template unknow(365), user config
7921
-- SEP001BD457F8B1: Phone available lines 1

Try a later build and see what happens then.

Regards,

Steve
