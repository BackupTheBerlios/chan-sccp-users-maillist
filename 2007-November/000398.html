With the help of a good friend, I managed to get past the problem. It
turns out that because I'm using a 64-bit machine for my Asterisk
server, there is a bug in the code in sccp_socket.c. I have submitted a
patch via the SourceForge bug tracker for this issue. Now the phone
registers fine and I can make and receive calls, however I am dealing
with a one-way audio problem.  The 7921G sends audio but will not
receive, no matter which way the call was initiated.
