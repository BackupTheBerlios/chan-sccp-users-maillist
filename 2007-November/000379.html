Hi all,

finally we have a release of chan-sccp-b which compiles out-of-the- 
box on both Asterisk 1.4 and 1.2 - you just have to type make to do  
so. There is not much of an updated documentation yet, so don't  
hesitate to ask here in the first place if you've got any questions  
as to the new features (which you probably are already using ;-).

Please check our project site on sourceforge to download it:

http://sourceforge.net/projects/chan-sccp-b/

We will soon move our codebase to the sourceforge svn server so you  
can stay even more up-to-date with what we're doing or even checkin  
some changes yourself. Please let us know if you'd like to be added  
as a project developer.

In the meantime we hope you enjoy this first "real" release and are  
anxious for feedback if it worked for you.
